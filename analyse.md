## 2. Bexio – Was uns noch fehlt ❌

### 🔴 Kritisch (fehlt im Kern-Workflow)

| Feature | Beschreibung |
|---|---|
| **Ausgangsrechnung erstellen** | Professionelle Rechnungen mit Briefkopf, Logo, Nummernkreis, QR-Rechnung generieren |
| **QR-Rechnung ausgeben** | Swiss QR-Code auf eigener Rechnung drucken (nicht nur einlesen) |
| **Kontaktverwaltung (CRM)** | Kunden- und Lieferantenstamm: IBAN, Adresse, UID, Kontaktperson |
| **Bankabgleich (Reconciliation)** | Bankkonto-Feed importieren, Buchungen auto-matchen oder manuell zuordnen |
| **ISO 20022 / pain.001 Export** | Zahlungsdatei für EBICS/E-Banking direkt exportieren |
| **Offizielle MwSt-Abrechnung (ESTV)** | Formular 533 / 583 ausfüllen und einreichen (nicht nur Übersicht) |

### 🟠 Wichtig (oft nachgefragt)

| Feature | Beschreibung |
|---|---|
| **Offerten / Angebote** | Angebotsstellung mit Umwandlung in Rechnung |
| **Automatische Mahnungen versenden** | Zahlungserinnerungs-E-Mail direkt aus App schicken (nicht nur Badge anzeigen) |
| **Mehrbenutzer + Rollen** | Admin / Buchhalter / Mitarbeiter, getrennte Zugriffsrechte |
| **Treuhänder-Zugang** | Externer Steuerberater kann Read-only-Zugriff erhalten |
| **Spesenabrechnungen** | Mitarbeiter-Spesen erfassen, genehmigen, verbuchen |
| **Mehrwährung (mit FX-Kursen)** | CHF, EUR, USD mit tagesaktuellem Wechselkurs + Kursdifferenzbuchung |
| **ELM-zertifizierter Lohnausweis** | SWISSDEC-konformer Lohnausweis, elektronisch übermittelbar (aktuell nur PDF-Export) |
| **Jahresabschluss-Assistent** | Geführte Abschlussbuchungen (Rückstellungen, Abschreibungen, Gewinnverwendung) |

### 🟡 Nice-to-have (mittelfristig)

| Feature | Beschreibung |
|---|---|
| **Mobile App** | iOS/Android-Native oder PWA für Belegscan unterwegs |
| **Zeiterfassung** | Stunden auf Projekte/Kunden buchen |
| **Anlagenbuchhaltung** | Anlagevermögen mit Abschreibungsplan (Sachanlagen, Fahrzeuge, etc.) |
| **Lagerverwaltung (minimal)** | Produkte/Artikel-Stamm für Rechnungspositionen |
| **Mehrsprachigkeit** | DE / FR / IT / EN (besonders für Westschweizer KMU) |
| **Kundenkonto-Portal** | Kunde kann eigene Rechnungen online einsehen und bezahlen |
| **10-Jahres-Archivierung (GeBüV-konform)** | Im todo.md vermerkt – gesetzlich vorgeschrieben |

## 3. Infinity.swiss – Was wir bereits decken ✅

Infinity positioniert sich als **AI-first Swiss Accounting Tool** – sehr ähnlich zu unserer Kernidee:

| Feature | Infinity | Bookitty |
|---|---|---|
| AI-gestützte Buchungsvorschläge | ✅ | ✅ (kittySuggester, 18+ Regeln) |
| Belegerkennung / OCR | ✅ | ✅ (Tesseract + QR-SPC) |
| Swiss Kontenrahmen KMU | ✅ | ✅ |
| Doppelte Buchführung | ✅ | ✅ |
| Modernes UI | ✅ | ✅ |
| Automatischer Beleg-Import | ✅ | ✅ (IMAP + Upload) |
| Buchungsvorschlag aus Dokument | ✅ | ✅ (documentParser + templateStore) |
| Swiss MwSt-Sätze (8.1 / 3.8 / 2.6%) | ✅ | ✅ |

---

## 4. Infinity.swiss – Was uns noch fehlt ❌

| Feature | Beschreibung |
|---|---|
| **Direkte Bankverbindung** | Kontoauszüge per Open Banking / API automatisch einlesen (PostFinance, Raiffeisen, ZAK) |
| **Automatisches Matching** | Kontoauszug-Zeile ↔ offene Buchung automatisch zuordnen |
| **Ausgangsrechnungen mit AI** | Rechnung auf Basis von Kundenhistorie vorschlagen und versenden |
| **Belegkategorisierung ohne Zutun** | Vollautomatisch (kein Klick nötig) statt nur Vorschlag – Einzel-Klick-Bestätigung |
| **Integriertes Zahlungssystem** | TWINT / Stripe / Kreditkarte als Zahlungslink in Rechnung einbetten |
| **Kollaboration / Team-Funktion** | Kommentare, Aufgaben, Genehmigungen auf Buchungen |

---

## 5. Unsere echten Stärken – was weder Bexio noch Infinity so hat

| Differenzierungs-Feature | Beschreibung |
|---|---|
| **Kitty KI-Chatbot** | Kontextbewusster Chat mit Buchungs-Preview + Direkterfassung in einem Klick |
| **Onboarding ohne Vorkenntnisse** | Interaktives Tutorial, kompletter Demo-Modus ohne Account |
| **Agentic UI** | Nav-Highlight via CustomEvent, Kitty hält Kontext aus Buchungsliste |
| **Steuerschätzung provisorisch** | Kantonales + Bundessteuer direkt sichtbar (Bexio hat das nicht integriert) |
| **Open Source / kein Lock-in** | Kein Abo-Modell, selbst hostbar |
